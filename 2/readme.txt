В файле HideAndSeek.php находится код метода подсчёта. Алгоритм базируется на однонаправленном списке.
В файле has.php находится код для работы с сервером. Код метода взят без изменений из файла HideAndSeek.php.

Для запуска метода локально использовать
>php HideAndSeek.php

Для запуска сервера использовать
>php -S localhost:8000

Для выполнения запроса к серверу использовать cURL с необходимыми параметрами. Пример команды представлен ниже.
>curl 'http://localhost:8000/has.php' -ntent-type: application/json" -d '{"children":["Ваня","Аня","Настя","Алена","Петя","Коля","Лена","Саша","Рома","Катя","Андрей","Максим"],"count": 4}'

Результат будет в виде строки в формате JSON вида {"name":"Ваня"}.
